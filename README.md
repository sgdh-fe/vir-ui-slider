## 安装

### npm

```sh
npm install vir-ui-slider
```

### script 标签

```html
<script src="https://unpkg.com/vir-ui-slider"></script>
<!-- 全局变量 VirUiSlider -->
```

## 基本使用

```js
const Slider = require('vir-ui-slider')()

new Slider({
  el: '.slider'
})
```
